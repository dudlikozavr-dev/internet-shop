# Mia-Amore Web Store (carstvo-sna.ru)

Веб-магазин для продажи женской домашней одежды бренда Mia-Amore.

## Структура проекта

```
internet-shop/
├── frontend/          (Next.js 14)
├── backend/          (FastAPI — часть mini-app/backend)
└── SPECIFICATION.md  (требования и архитектура)
```

## Quick Start

См. SPECIFICATION.md для полной информации о требованиях, архитектуре и плане реализации.

## Стек

- **Frontend:** Next.js 14, TypeScript, Tailwind CSS, shadcn/ui
- **Backend:** FastAPI (расширение существующего из mini-app)
- **Database:** PostgreSQL на timeweb VPS
- **Storage:** Cloudinary
- **Payments:** YooKassa
- **Delivery:** CDEK, Почта России, Яндекс.Доставка, Курьер Москва
- **Email:** Unisender/Mindbox
- **Analytics:** Yandex.Metrika
